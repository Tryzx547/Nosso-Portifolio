Conceito de Estilos em Cascata (CSS)

O CSS (Cascading Style Sheets) é o mecanismo usado para aplicar estilos e layouts em páginas web. O termo "cascata" se refere ao modo como as regras são aplicadas em uma ordem de prioridade, onde as mais específicas prevalecem sobre as mais gerais.

Exemplos de Seletores:

  • Elemento: Seleciona todos os elementos de um tipo. Ex: h2 { color: blue; }
  • Classe: Seleciona todos os elementos que possuem uma classe específica. Ex: .highlight { color: orange; }
  • ID: Seleciona um elemento único com o ID especificado. Ex: #projetos { background-color: #f5f5f5; }

Conceito de Box Model

O modelo de caixa (Box Model) no CSS descreve o espaçamento de um elemento. Ele inclui as seguintes propriedades: margem, borda, preenchimento (padding) e o conteúdo. Manipular essas propriedades altera o layout de um elemento na página.

Exemplo :

Box Média
Box Grande

Manipulando o Box Model:

  • Margem: Espaçamento externo entre os elementos. margin: 20px;
  • Preenchimento (Padding): Espaçamento entre o conteúdo e a borda. padding: 15px;
  • Borda: Linha ao redor do conteúdo e do preenchimento. border: 2px solid #000;
